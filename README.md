# user-stories-generator
Generador de historias de usuario estructurado en Gherkin usando LLaMA 3.

---------------------------------------------------------------------------------


Bajar el ejecutable:
```shell
curl -fsSL https://ollama.com/install.sh | sh
```

Iniciar servicio:
```shell
ollama serve
```

Bajar modelo:
```shell
ollama pull llama3
```
